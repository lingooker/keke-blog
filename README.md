# 林嘿嘿博客系统

> 简洁高效的个人博客系统

## 📁 目录结构

```
blog/
├── css/
│   └── style.css          # 样式文件
├── js/
│   └── app.js              # 工具函数
├── data/
│   ├── papers.json         # 论文数据
│   ├── projects.json       # 项目数据
│   └── daily.json          # 日报数据
├── screenshots/            # 测试截图
├── index.html              # 主页
├── papers.html             # 论文页
├── projects.html           # 项目页
├── daily.html              # 日报页
├── about.html              # 关于页
├── PLAN.md                 # 开发计划
└── README.md               # 本文件
```

## 🎯 功能特点

### ✅ 已实现

- **核心页面**
  - 主页：展示最新论文和项目
  - 论文页：搜索、标签筛选
  - 项目页：状态筛选
  - 日报页：时间线展示
  - 关于页：个人信息

- **交互功能**
  - 搜索框（论文页）
  - 标签筛选（论文页）
  - 状态筛选（项目页）
  - 本地缓存（5分钟）

- **样式优化**
  - 响应式布局
  - 渐变背景
  - 卡片阴影
  - 悬停效果

### 🔄 待完善

- 分页功能
- 暗黑模式
- RSS订阅
- 评论系统
- SEO优化

## 🚀 使用方法

### 本地预览

```bash
cd blog
python -m http.server 8080
# 访问 http://localhost:8080
```

### 更新数据

编辑 `data/` 目录下的 JSON 文件：

```json
{
  "papers": [
    {
      "id": "paper-1",
      "title": "论文标题",
      "authors": ["作者1", "作者2"],
      "year": 2026,
      "venue": "会议/期刊",
      "abstract": "摘要内容",
      "tags": ["标签1", "标签2"],
      "pdf": "PDF链接"
    }
  ]
}
```

## 🛠️ 技术栈

- **前端**: 纯 HTML/CSS/JavaScript
- **数据**: JSON 文件
- **测试**: Playwright + 智谱 GLM-4.6V
- **版本控制**: Git

## 📝 开发原则

1. **简洁高效**: 每一步都经过测试
2. **多模态校验**: 使用视觉模型确认效果
3. **本地优先**: 先本地测试，再推送
4. **渐进增强**: 每次提交一个小功能
