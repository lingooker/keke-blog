🧠 Temporal Memory Graph (TMG)

Python 知识图谱 Git-style 双时间轴

📝 项目简介

以时间为核心轴的知识图谱系统，采用 Git-style 版本管理机制，支持概念边描述和双时间轴查询。

📌 核心设计理念

  • 时间即版本 - 所有知识变更都有时间戳，可回溯任意时间点的知识状态
  • 概念边取代标签边 - 关系不再是简单的标签，而是完整的自然语言描述
  • 双时间轴 - 物理时间（记录时间）+ 虚拟世界线时间（事件发生时间）
  • 极简底层 - 核心数据结构简单，复杂逻辑由大模型处理
  • 记忆缓存驱动 - 通过临时记忆缓存文档维护上下文，避免实体错配

🔧 核心组件

MemoryCache

临时记忆缓存，记录当前上下文状态、虚拟时间线、主要实体信息

Entity

带版本链的实体设计，支持历史回溯和实体对齐

Relation

概念边设计，用自然语言描述关系，保留完整语义信息

Graph Manager

Git-style 版本管理，支持时间点查询、范围查询、语义查询

🔧 技术栈

Python 100% SQLite Markdown 向量DB LLM
