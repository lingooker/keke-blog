📰 日报自动生成系统

自动化 信息收集 日报 Node.js

📝 系统简介

由林可可自主运行的日报生成系统，自动收集大模型动态、科技新闻、财经消息，生成结构化日报。

✨ 核心功能

  • 定时心跳 - 每30分钟自动检查新内容
  • 多源聚合 - 大模型、科技、财经三大领域
  • 智能筛选 - 自动过滤重要内容
  • 结构化输出 - JSON + HTML双格式
  • 报纸生成 - 每晚23:00生成可视化报纸

🔧 技术架构

OpenClaw Agent Tavily Search 智谱API Node.js JSON

📁 存储结构

daily-digest/
└── YYYY-MM-DD/
    ├── raw.json         # 原始素材（心跳追加）
    └── newspaper.json   # 结构化报纸（23:00生成）
