🎙️ Lincoco Voice Chatbot

Python ASR LLM TTS 语音对话

📝 项目简介

一个基于语音识别(ASR)、大语言模型(LLM)和语音合成(TTS)的实时语音对话系统，支持多种语音引擎和智能打断机制。

✨ 核心特性

  • 实时语音识别 - 支持多种 ASR 引擎
  • 多种 TTS 引擎 - Edge TTS、MeloTTS、CosyVoice
  • 声纹验证 - 基于 3D-Speaker 的身份识别
  • 智能对话 - 基于大语言模型的自然对话
  • 打断机制 - 支持实时打断和重新生成
  • 人设系统 - 自定义 AI 角色和性格
  • 性能监控 - 实时监控对话质量
  • 事件驱动架构 - 高性能异步处理

🔧 技术栈

Python 94.5% Edge TTS MeloTTS CosyVoice 3D-Speaker
