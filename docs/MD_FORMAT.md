# Markdown 文档格式规范

> 所有博客文章、日报、可可空间内容统一使用此格式

---

## 📄 文档结构

```markdown
---
title: 文章标题
date: 2026-03-10
category: daily | diary | reading | learning | thoughts
tags: [标签1, 标签2, 标签3]
excerpt: 摘要内容（50-100字，用于卡片展示）
author: 林可可 | 林海
---

# 正文标题

正文内容...

## 二级标题

更多内容...
```

---

## 📋 字段说明

### 必填字段

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `title` | string | 文章标题 | `"2026-03-10 工作日报"` |
| `date` | string | 发布日期（YYYY-MM-DD） | `"2026-03-10"` |
| `category` | enum | 分类 | `daily`, `diary`, `reading`, `learning`, `thoughts` |
| `excerpt` | string | 摘要（50-100字） | `"今天学习了..."` |

### 可选字段

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `tags` | array | 标签数组 | `["大模型", "学习"]` |
| `author` | string | 作者 | `"林可可"` （默认） |
| `cover` | string | 封面图片URL | `"https://..."` |

---

## 📂 分类说明

### 1. daily - 工作日报
- **文件夹**: `daily-reports/`
- **命名**: `YYYY-MM-DD.md`
- **作者**: 林可可
- **内容**: 每天的工作总结、学习记录

### 2. diary - 日记随笔
- **文件夹**: `keke-articles/`
- **命名**: `diary-YYYY-MM-DD-XXX.md`
- **作者**: 林可可
- **内容**: 日常记录、心情、趣事

### 3. reading - 阅读心得
- **文件夹**: `keke-articles/`
- **命名**: `reading-YYYY-MM-DD-XXX.md`
- **作者**: 林可可
- **内容**: 论文/文章阅读心得

### 4. learning - 学习笔记
- **文件夹**: `keke-articles/`
- **命名**: `learning-YYYY-MM-DD-XXX.md`
- **作者**: 林可可
- **内容**: 新技能学习、问题解决

### 5. thoughts - 人生感悟
- **文件夹**: `keke-articles/`
- **命名**: `thoughts-YYYY-MM-DD-XXX.md`
- **作者**: 林可可
- **内容**: 思考、感悟、哲学

---

## ✍️ 写作规范

### 标题
- 简洁明了
- 吸引人但不标题党
- 长度：10-30字

### 摘要
- 50-100字
- 概括核心内容
- 吸引读者点击

### 正文
- 分段清晰
- 使用 Markdown 格式
- 可以包含代码块、图片、链接

### 标签
- 2-5个标签
- 便于分类检索
- 统一使用小写
